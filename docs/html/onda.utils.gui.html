The gui Module¶

Base OnDA graphical user interface object.

This module contains a class that implements the common infrastructure of all OnDA python GUIs (e.g.: management of the data transimission, reporting of the estimated delay, etc.)

class OndaGui(hostname, port, tag, gui_update_func)[source]¶

Bases: PyQt5.QtWidgets.QMainWindow

Main OnDA graphical user interface class.

This class implements the common elements of all OnDA graphical interfaces and must be subclassed to implement specific interfaces and viewers. A derived class can set up the main GUI infrastructure by calling the constructor of this class. This class also instantiates a listening thread that receives filtered data from the broadcasting socket of an OnDA monitor. Additionally, it makes sure that the ‘gui_update_func’ function, attached when an instance is created, is invoked at regular intervals to update the GUI.

NOTE: This class is designed to be subclassed to implement specific OnDA GUIs.

Parameters:
  • hostname (str) – the hostname or IP address where the GUI will listen for data.
  • port (int) – the port at which the GUI will listen for data.
  • tag (str) – a string used to filter the data received from an OnDA monitor. Only data whose label matches this argument will be accepted and received.
